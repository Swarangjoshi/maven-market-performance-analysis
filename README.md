# maven-market-performance-analysis
Interactive Power BI dashboard analyzing Maven Market sales, orders, returns, customers, stores, and products using DAX and Power Query.


Maven Market Performance Dashboard
Project Overview

This project presents a comprehensive multi-page Power BI dashboard built using the Maven Market dataset. The dashboard provides valuable insights into sales, orders, returns, customers, stores, and products to support business decision-making and performance optimization.

Dashboard Pages
1. Sales Analysis
Sales trends by year and month
Sales performance by region
Top-selling products
Product brand contribution to revenue
2. Order Overview
Total orders and quantity sold
Order trends over time
Store-wise order performance
Relationship between orders and sales
3. Return Analysis
Return quantity analysis
Return trends by product and region
Impact of returns on business performance
4. Customer Insights
Customer segmentation by education, occupation, and membership card
High-value customer identification
Customer demographics analysis
5. Stores Performance
Store-wise sales comparison
Regional store performance
Store growth analysis
6. Products Analysis
Product profitability analysis
Top and low-performing products
Product performance across regions
